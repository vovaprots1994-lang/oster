// Інструкція до шаблону

functions\ - папка використовуєтсь для підключення додатковий функцій, класів, бібліотек 
-- admin.php - пишемо функції які відповідають за розширення функціоналу адмін-панелі
-- taxonomy.php - використовуємо для реєстрацій кастомних посттайпів і таксономій
-- class-aq_resize.php - клас роботи з зображеннями

templates\ - використовуємо для файлів які відповідають за вивід кастомних сторінок
-- about.php - приклад файлу для вивову кастомної сторінки

style.css - основний файл стилів, використовується для опису шаблону і додаткових стилів

README.md - опис шаблону, видалити для при використанні

404.php - вивід 404 сторінки

front-page.php - використовувати для виводу головної сторінки, не є обов'язковим

functions.php - основні функції шаблону, підключення скритів і стилів. ВАЖЛИВО. функції для кастомізації фронтенду (хлібні крихти, вивід тумсів новни тощо) писати в окремий файл functions\frontend.php

index.php - файл використовуэться для виводу новин. не використовувати для головної сторінки чи інших типів

page.php - вивід стандартної сторінки. Обов'язковий файл

screenshot.png - зображення теми, використовуємо скріншот головної сторінки або логотип проекту
